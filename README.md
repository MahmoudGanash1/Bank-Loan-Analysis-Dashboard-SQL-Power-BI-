# 🏦 Bank Loan Analysis Dashboard (SQL + Power BI)

An interactive analytics project designed to explore **bank loan performance, approval trends, and financial behavior patterns** using SQL and Power BI.

This project demonstrates how combining **SQL for data extraction** and **DAX for advanced calculations** can deliver meaningful business insights through interactive dashboards.

---

## 🎯 BUSINESS OBJECTIVE

The goal of this project is to analyze bank loan data in order to:
- Understand loan approval and rejection patterns  
- Track key financial KPIs  
- Identify risk and performance trends  
- Support data-driven financial decision-making  

---

## 💡 SOLUTION

A complete end-to-end analytics workflow was built using:
- SQL for data extraction, cleaning, and preparation  
- Power BI for visualization and dashboard development  
- DAX for advanced KPI calculations and dynamic measures  

---

## 🛠 TOOLS & TECHNOLOGIES

- SQL (Data Extraction & Preparation)  
- Power BI (Data Visualization & Dashboarding)  
- DAX (Advanced Calculations & KPIs)  
- Data Modeling  

---

## 📊 KEY FEATURES

- Loan performance analysis (approved vs rejected)  
- Financial KPI tracking using DAX measures  
- Customer and loan segmentation  
- Risk and trend analysis  
- Interactive filters and drill-down capabilities
